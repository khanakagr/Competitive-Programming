# 1. The Parity Duel 

### Description
**Evenor** and **Oddrick** are bitter rivals playing a number game. 
- The game starts with an integer $n$.
- **Evenor** moves first.
- On **Evenor's** turn, he must choose an **even** integer $k$ ($1 \le k \le n$) and subtract it from $n$.
- On **Oddrick's** turn, he must choose an **odd** integer $k$ ($1 \le k \le n$) and subtract it from $n$.
- The player who cannot make a valid move loses the duel.

Assume both play optimally. Determine the winner.

### Input
A single integer $n$ ($1 \le n \le 10^9$).

### Output
Print "Evenor" if he wins, otherwise print "Oddrick".

### Sample
**Input:**
`1`
**Output:**
`Oddrick`
