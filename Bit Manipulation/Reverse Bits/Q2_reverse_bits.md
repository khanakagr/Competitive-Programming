# 2. Reverse Bits 

### Description
Reverse the bits of a given 32-bit unsigned integer. Note that for this problem, you must treat the input as a string of 32 bits, including leading zeros.
---

### Input
- A 32-bit unsigned integer represented as a decimal.

**Constraints:**
- The input is a 32-bit integer.

---

### Output
Print the decimal value of the resulting reversed 32-bit integer.

---

### Sample
**Input:**
43261596

**Note:** The input binary `00000010100101000001111010011100` becomes `00111001011110000010100101000000`.

**Output:**
964176192
